# Condicionales y Bucles en JavaScript

## Condicionales
Permiten ejecutar bloques de código solo si se cumple una condición específica.
-   **If / Else:** La forma más básica. Si `a` es verdad, haz esto; sino, haz aquello.
-   **Switch:** Útil cuando comparamos una misma variable contra múltiples valores posibles.
-   **Ternario (`? :`):** Una forma concisa de escribir un `if...else` en una sola línea.

## Bucles (Loops)
Permiten repetir una acción múltiples veces.
-   **For:** El bucle más común. Tienes control total sobre el inicio, la condición y el paso.
-   **While:** Repite mientras la condición sea `true`. Cuidado con los bucles infinitos.
-   **Do...While:** Similar al `while`, pero garantiza que el código se ejecute al menos una vez antes de verificar la condición.
-   **For...Of:** Moderno y limpio, ideal para recorrer Arrays.
-   **For...In:** Diseñado para recorrer las propiedades (claves) de un Objeto.
