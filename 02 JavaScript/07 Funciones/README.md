# Funciones en JavaScript

Las funciones son bloques de código reutilizables diseñados para realizar una tarea específica.

## Tipos de Funciones

1.  **Declaración de Función (Declaration):** Usan la palabra clave `function`. Tienen *hoisting*, por lo que pueden invocarse antes de su definición en el código.
2.  **Expresión de Función (Expression):** Se asignan a una variable. No tienen hoisting.
3.  **Funciones Flecha (Arrow Functions):** Introducidas en ES6. Tienen una sintaxis más limpia y no vinculan su propio `this` (heredan el contexto).

## Parámetros
Las funciones pueden recibir datos de entrada llamados parámetros. Desde ES6, podemos asignar **parámetros por defecto** (`param = valor`).

## Return
La palabra clave `return` finaliza la ejecución de la función y especifica un valor para ser devuelto a quien llamó a la función.
