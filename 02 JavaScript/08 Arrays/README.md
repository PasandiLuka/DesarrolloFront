# Arrays (Arreglos) en JavaScript

Un Array es una variable especial que puede contener más de un valor a la vez. Es una lista ordenada de datos.

## Métodos Comunes
JavaScript ofrece muchos métodos incorporados para trabajar con arrays:

### Modificar el Array Original (Mutables)
-   `push()`: Agrega al final.
-   `pop()`: Elimina del final.
-   `unshift()`: Agrega al inicio.
-   `shift()`: Elimina del inicio.
-   `splice()`: Añade o elimina elementos en cualquier posición.

### Crear Nuevos Arrays (Inmutables)
-   `slice()`: Retorna una copia de una parte del array.
-   `concat()`: Une dos o más arrays.

### Iteración y Transformación (ES6+)
-   `forEach()`: Ejecuta una función para cada elemento.
-   `map()`: Crea un nuevo array con los resultados de llamar a una función para cada elemento.
-   `filter()`: Crea un nuevo array con todos los elementos que cumplan una condición.
-   `find()`: Devuelve el primer valor que cumple la condición.
