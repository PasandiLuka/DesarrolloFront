# Documentación HTML

Este repositorio contiene ejemplos prácticos y documentación sobre diferentes aspectos del desarrollo frontend con HTML. A continuación, encontrarás un índice del material disponible en cada directorio.

## Índice de Contenidos

### [00 Estructura Base](./00%20EstrucuturaBase/README.md)
Aprende sobre la estructura fundamental de un documento HTML5, incluyendo `<html>`, `<head>`, `<body>` y metadatos importantes.

### [01 Etiquetas de Texto](./01%20Etiquetas/README.md)
Explora las etiquetas para dar formato al texto (`<p>`, `<b>`, `<i>`), encabezados (`<h1>`-`<h6>`) y enlaces (`<a>`).

### [02 Contenido Multimedia](./02%20ContenidoMultimedia/README.md)
Descubre cómo integrar imágenes (`<img>`), videos (`<video>`), audio (`<audio>`) y gráficos vectoriales (`<svg>`).

### [03 Listas](./03%20Listas/README.md)
Ejemplos de listas ordenadas (`<ol>`) y desordenadas (`<ul>`), así como su anidación.

### [04 Tablas](./04%20Tablas/README.md)
Aprende a estructurar datos en filas y columnas utilizando `<table>`, `<tr>`, `<td>`, `<th>` y cómo agrupar columnas.

### [05 Agrupadores](./05%20Agrupadores/README.md)
Diferencias y usos de los contenedores `<div>` (bloque) y `<span>` (línea).

### [06 IFrame](./06%20IFrame/README.md)
Cómo incrustar contenido externo como videos de YouTube, mapas de Google Maps y otras páginas web mediante `<iframe>`.

### [07 Formularios](./07%20Formularios/README.md)
Guía completa sobre la creación de formularios con `<form>`, diversos tipos de `<input>`, validaciones y otros controles como `<select>` y `<textarea>`.
