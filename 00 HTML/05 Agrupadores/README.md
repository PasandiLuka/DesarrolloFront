# Agrupadores

Este directorio explica las etiquetas utilizadas para agrupar otros elementos HTML y aplicar estilos o scripts de manera conjunta.

## `<div>` (División)

La etiqueta `<div>` define una división o sección en un documento HTML.
*   **Comportamiento**: Es un elemento de **bloque**. Esto significa que ocupa todo el ancho disponible y comienza en una nueva línea.
*   **Uso**: Se utiliza comúnmente como contenedor para otros elementos HTML (títulos, párrafos, imágenes, etc.) para estructurar el diseño de la página (layout) y aplicar estilos CSS a grupos de elementos.

## `<span>` (Abarcar)

La etiqueta `<span>` se utiliza para agrupar contenido en línea.
*   **Comportamiento**: Es un elemento de **línea**. Esto significa que solo ocupa el ancho necesario para su contenido y *no* comienza en una nueva línea.
*   **Uso**: Se utiliza para aplicar estilos o manipular una parte específica de un texto o contenido dentro de un elemento de bloque (como dentro de un párrafo), sin alterar el flujo del documento.
