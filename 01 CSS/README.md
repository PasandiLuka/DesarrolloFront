# Documentación CSS

Este repositorio contiene ejemplos prácticos y documentación sobre diferentes aspectos de CSS3.

## Índice de Contenidos

### [00 Sintaxis Básica](./00%20SintaxisBasica/index.html)
Introducción a cómo vincular CSS y selectores básicos.

### [01 Selectores](./01%20Selectores/index.html)
Selectores de etiqueta, clase, id y universal.

### [02 Background](./02%20Background/index.html)
Fondos, colores, imágenes y gradientes.

### [03 Bordes](./03%20Bordes/index.html)
Estilos de bordes, redondeo y grosores.

### [04 Unidades](./04%20Unidades/README.md)
Unidades de medida absolutas (px) y relativas (rem, em, %, vw, vh).

### [05 Texto y Fuentes](./05%20TextoFuentes/README.md)
Tipografía, alineación, decoración y peso de fuentes.

### [06 Modelo de Caja](./06%20ModeloCaja/README.md)
Padding, Margin, Border y Box-Sizing.

### [07 Listas y Tablas](./07%20ListasTablas/README.md)
Estilización de listas (ul, ol) y tablas (bordes, zebra-striping).

### [08 Display y Position](./08%20DisplayPosition/README.md)
Propiedades de visualización (block, inline) y posicionamiento (relative, absolute, fixed, sticky).

### [09 Flexbox](./09%20Flexbox/README.md)
Diseño flexible unidimensional. Alineación y distribución.

### [10 Grid](./10%20Grid/README.md)
Diseño de cuadrícula bidimensional. Columnas, filas y áreas.

### [11 Animaciones](./11%20Animaciones/README.md)
Transiciones, transformaciones y animaciones con @keyframes.
