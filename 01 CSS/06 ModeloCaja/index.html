El Modelo de Caja (Box Model)

Contenido (Content)

Margin: Espacio externo (naranja).

Border: Borde del elemento (negro).

Padding: Espacio interno (verde).

Content: El contenido real (azul).

Box Sizing

box-sizing: content-box (Default)
Width = 200px + padding + border
box-sizing: border-box
Width = 200px (incluye padding y border)
