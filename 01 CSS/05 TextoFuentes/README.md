# Texto y Fuentes en CSS

CSS nos permite controlar la apariencia del texto de muchas formas, desde la tipografía hasta la alineación y decoración.

## Propiedades Principales

### Tipografía (Font)
- **font-family:** Define la familia de fuentes a usar (ej. Arial, Times New Roman). Se suelen poner varias separadas por comas como "fallback".
- **font-size:** Tamaño de la fuente (px, rem, em).
- **font-weight:** Grosor de la fuente (normal, bold, 100-900).
- **font-style:** Estilo de la fuente (normal, italic).

### Texto (Text)
- **text-align:** Alineación horizontal del texto (left, center, right, justify).
- **text-decoration:** Decoraciones como subrayado o tachado (underline, line-through, none). Muy usado para quitar la línea a los enlaces.
- **text-transform:** Cambiar a mayúsculas o minúsculas (uppercase, lowercase, capitalize).
- **line-height:** Altura de la línea, útil para mejorar la legibilidad de párrafos.

## Fuentes Externas
Es común usar fuentes externas como Google Fonts importándolas en el `<head>` o vía `@import` en CSS para tener tipografías más modernas y variadas.
